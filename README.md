# FIFA World Cup 2026 Smart Robot Management System

An AI-powered Smart Stadium Robot Management Dashboard built using Python, Streamlit, Pandas and Plotly . The system simulates autonomous robots operating inside a FIFA World Cup stadium and provides real-time monitoring, analytics, mission control, predictive insights and emergency alerts.



# Project Overview

Managing hundreds of autonomous robots inside a smart stadium requires continuous monitoring and intelligent decision-making.

This project provides a centralized dashboard that monitors robot activity, battery levels, task allocation, movement, emergency alerts and AI-powered recommendations in real time.

The dashboard automatically refreshes every few seconds to simulate a live robotic environment.
